ТЗ Личный кабинет
Задача: Реализовать экран личного кабинета пользователя.

Описание:
Создать экран "Личный кабинет". 
Отобразить на экране захардкоженные основные данные пользователя, Истории заказов и тд. (согласно дизайну).
Реализовать возможность редактирования данных пользователя, и удаления аккаунта.
Визуальное представление личного кабинета должно соответствовать дизайну в Figma. Figma Если отдельные элементы требуют перехода на другие трудоемкие экраны (например Корзину или Детали заказа) - ставьте заглушки. Упор на функционал внутри Личного кабинета.
